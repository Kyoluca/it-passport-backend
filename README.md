# IT Passport 中文智能备考系统 - 后端数据流水线 🚀

本项目是在日华人 IT Passport 备考网站的自动化数据清洗与解析引擎。通过大模型（LLM）双文件联动技术，实现官方双栏 PDF 真题的自动化结构化提取、精准中文翻译及体系化考点归纳。

## 🛠️ 当前技术栈
- **Language:** Ruby, Bash
- **AI Engine:** Gemini API (3.1 Lite / 1.5 Flash)
- **Data Source:** 日本独立行政法人情报处理推进机构 (IPA) 官方公开题

## ✨ 核心功能 (MVP V3.1)
- [x] **双文件联动校验**：同时读取真题 PDF 与标准答案 PDF，确保 AI 生成的解析核心逻辑与官方答案 100% 一致。
- [x] **高并发分块批处理**：自动将 100 道题切分为多个批次进行增量提取，完美避开大模型单次输出字数上限。
- [x] **工业级容错机制**：内置自动重试逻辑，智能应对网络超时及服务器高并发拥堵。
- [x] **自动化流水线**：一键执行 Bash 脚本批量下载历年真题，Ruby 引擎自动接管清洗入库。
